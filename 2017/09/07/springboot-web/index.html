restful and thymeleaf

springboot restful 架构,是springboot快速发布的web架构，用于rpc调用和请求。
springboot thymeleaf 架构，是springboot-thymeleaf的快速web架构，用于请求html界面展示。

restful架构在Controller 使用 @RestController注解
调用方法中使用 @RequestMapping(“/saveuser”) 返回json或其他值即可

thymeleaf架构是在Controller 使用 @Controller 注解
@RequestMapping(“/index”)
public String greeting() {
return “index”;
}
配置文件需要配置：

##thymeleaf start
spring.thymeleaf.mode=HTML5
spring.thymeleaf.encoding=UTF-8
spring.thymeleaf.content-type=text/html

##开发时关闭缓存,不然没法看到实时页面
spring.thymeleaf.cache=false

##thymeleaf end 即可
